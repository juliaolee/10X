
##  How to Run

Clone the repository: 
```bash
git clone https://github.com/juliaolee/10X.git
```
Open index.html in your browser.

### For development:

Install VSCode extensions: Live Sass Compiler and Live Server.

Run Live Server on index.html.

Watch SCSS for auto-compilation to CSS.
